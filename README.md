# my-flappy-bird
Jogo desenvolvido no Curso de Libgdx da Udemy para dispositivos Android. 

Também compatível com Desktop/Navegador Web.

![my-piano-tiles](imgs/my-flappy-bird.gif)  
Jogue o demo do jogo [clicando aqui](https://safe-depths-05400.herokuapp.com/).

## Como jogar
* Clique com o mouse _(ou toque a tela)_ várias vezes para controlar o vôo da ave.
* Atravesse as passagens pelos canos sem bater.
* Se bater nos canos, toque no botão no centro da tela para reiniciar.

## Instalar versão Android
* Fazer o download e instalação do APK através de seu dispositivo móvel, [clicando aqui](https://bit.ly/3sXWiKA).
